# Assignment Similarity System рҹҡҖ

A FastAPI-based backend system for detecting similarity between assignments.

---

## рҹӣ  Tech Stack

- FastAPI
- Uvicorn
- Pydantic
- JWT Authentication (python-jose)
- Passlib (bcrypt)
- Swagger UI (Auto-generated)

## рҹ“ҳ What is Swagger?

Swagger UI is an interactive documentation tool automatically generated by FastAPI.

It allows developers to test API endpoints directly in the browser without using Postman.


Access Swagger UI at:
http://127.0.0.1:8000/docs

You can:
- View all endpoints
- Test API requests
- Use JWT authentication
- See request & response schemas
